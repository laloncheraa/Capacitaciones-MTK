# 📊 Capacitaciones-MTK

Dashboard interactivo para el análisis de capacitaciones de técnicos utilizando Python y Streamlit.

---

## 🚀 Descripción

Este proyecto permite visualizar el progreso de capacitación de técnicos a partir de datos provenientes de Excel o SQL.

Incluye:

* KPIs de avance
* Filtros interactivos (Región, Instructor, Categoría)
* Visualización de estatus (Aprobado, Reprobado, Pendiente)
* Gráficas dinámicas

---

## 🧠 Tecnologías utilizadas

* Python
* Streamlit
* Pandas
* Excel / SQL (fuente de datos)

---

## 📁 Estructura del proyecto

```
Capacitaciones-MTK/
│
├── app.py              # Dashboard principal
├── requirements.txt    # Librerías necesarias
├── data.xlsx           # Base de datos (opcional)
```

---

## ⚙️ Instalación

1. Clona el repositorio:

```
git clone https://github.com/laloncheraa/Capacitaciones-MTK.git
cd Capacitaciones-MTK
```

2. Instala dependencias:

```
pip install -r requirements.txt
```

---

## ▶️ Ejecución

Corre el siguiente comando:

```
streamlit run app.py
```

Se abrirá automáticamente en tu navegador:

```
http://localhost:8501
```
